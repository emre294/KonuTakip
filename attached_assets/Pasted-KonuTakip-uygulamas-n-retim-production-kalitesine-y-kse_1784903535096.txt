KonuTakip uygulamasını üretim (production) kalitesine yükselt.

AMAÇ:
Premium üyelik alan kullanıcı uygulamaya girdiği anda gerçekten özel hissetsin. Arayüz modern, akıcı ve premium görünmeli. Aynı zamanda uygulamadaki tüm sayfalarda kullanıcı deneyimini iyileştir.

==================================================
GENEL KURALLAR
==================================================

- Değiştireceğin tüm dosyaların önce zaman damgalı yedeğini al.
- Geniş kapsamlı encoding düzeltmesi yapma.
- Gereksiz dosyaları değiştirme.
- Mevcut çalışan sistemi bozma.
- Kod tekrarından kaçın.
- Performansı düşürmeyecek animasyonlar kullan.
- Expo / React Native uyumlu çözüm üret.
- TypeScript hatası bırakma.
- İşlem sonunda typecheck çalıştır.
- Git diff göster.
- Hata oluşursa otomatik geri yükleme yap.
